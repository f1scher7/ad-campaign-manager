# Ad Campaign Manager

The Ad Campaign Manager was created using Spring Boot and Angular. It is a web application that allows administrators to manage marketing campaigns and users.

## Technologies

### Front-End
- Angular 18
- TypeScript
- HTML5
- CSS3
- Bootstrap5
- Font Awesome

### Back-End
- Java
- Spring Boot
- Spring Web MVC
- Spring Security
- JWT
- Maven
- Docker Image
- AWS EC2
- H2
- Spring Data JPA
- Hibernate
